<div align="center">
  <h1>Go2V</h1>
  <b>Go to V source code translator</b>
</div>

## Installation
```
git clone https://github.com/vlang/go2v
cd go2v
v run .
```

## Usage

Convert a file using `v run . file.go`.

See `v run . help` for more information.
